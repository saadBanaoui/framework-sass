Introduction

Un framework Css conçue pour almerys

almerysCss est un mini framework CSS de base, il a pour but l'harmonisation des projets d'intégration web au sein d'almerys, conçue par Saad EL BANAOUI et l'équipe Webdesign. Le framework regroupe un ensemble de bonnes pratiques, de conventions de nommage, d'astuces et de bouts de code "CSS" réutilisables.

Voici une liste des fonctionnalités que l'on trouvera dans almerysCss :

  • Reset
  • des classes réutilisables
  • des grilles simples et des gouttières
  • des aides au positionnement
  • du Responsive webdesign

Installation

Il suffit d'appeler la feuille de style "style.css" dans la section "head" .

<link rel="stylesheet" href="style.css" media="all">

Structure de base

			
			
				<div class="container">
    			          <div class="row">
    			            <div class="col12">
       			                   …
    			          </div>
    			      </div>
